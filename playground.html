Quick Setup

Prerequisites
Widget must be built and Next.js agent must be running

Build Svelte Widget

$ cd svelte-wrapper
$ npm run build

Builds virtual-agent.js and svelte-wrapper.css to svelte-wrapper/static/

Start Next.js Agent

$ cd nextjs-agent
$ npm run dev

Open Playground

The playground loads files directly from svelte-wrapper/static/

Widget Controls

Test different configurations in real-time

Position

Change widget position on screen

Console Commands

Try these commands in your browser console

// Change position VirtualAgent.updateConfig({ position: 'bottom-left' }); // Change color VirtualAgent.updateConfig({ primaryColor: '#8b5cf6' }); // Control visibility VirtualAgent.open(); VirtualAgent.close();
